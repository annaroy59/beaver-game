# Бобёр 🦫

Snake-подобная игра в браузере с аниме-бобром в стиле Наруто.

## Возможности

- **Движение**: стрелки, WASD, Shift (вверх), пробел (старт/автоигра), Enter (пауза)
- **Мобильное управление**: свайп по полю, D-pad кнопки, тап для старта
- **Телепорты**: Portal-стиль, голубой + оранжевый, автоматический респавн каждые 10 сек
- **Кукуруза-щит**: 15% шанс/тик, автоматический уворот от 1 столкновения
- **Магнит**: притягивает еду до 3 клеток, 10 сек, таймер
- **Автоигра (ИИ)**: берёт бонусы, использует телепорты, лимит 1 телепорт/тик
- **3D-рендеринг**: градиенты, тени, блики на всех элементах
- **Лазерные глаза**: злой бобёр стреляет лазерами на курсор мыши
- **Эмоции**: счастливый (победа + фейерверки), грустный (проигрыш + слёзы), злой (2+ проигрыша)
- **Глаза следят за мышью** на экранах game over, паузы и в попапах
- **Настройки**: скорость (1-10), автоускорение, победа по длине, размер сетки (10-16), телепорты on/off
- **Отзывы**: 21 отзыв, фильтр по версии, пагинация
- **Хелп**: карусель с анимированными canvas-демо

## Запуск

Откройте `index.html` в браузере.

## Тесты

```bash
npm install
node tests/test.js
```

90 тестов / 165 assertions (jsdom + canvas).

## Структура

```
index.html           — точка входа
src/game.js          — вся логика игры (~2480 строк)
src/styles.css       — стили
tests/test.js        — тесты
reviews/reviews.js   — данные отзывов
.mimo/               — контекст проекта для агентов
```

## Стек

Vanilla JS, CSS, Canvas. Без фреймворков.
