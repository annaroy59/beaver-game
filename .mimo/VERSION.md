# Текущая версия

## Фичи
- Движение: стрелки, WASD, Shift (вверх)
- Пробел: старт / автоигра (toggle)
- Enter: пауза
- Разворот на 180° с 1 сегментом
- Еда: брёвна, скорлупа +1 каждые 3
- Настройки: скорость (1-10), авто-ускорение, победа по длине, телепорты on/off
- Телепорты в стиле Portal: овальные кольца, голубой + оранжевый, свечение, вращающиеся дуги
- Кукуруза-щит: 15% шанс/тик, щит на 1 столкновение (авто-уворот)
- Магнит: 15% шанс/тик, притягивает бревно до 3 клеток, 10 сек, вишенка-пиктограмма, летающий красный шарик
- ИИ: берёт бонусы, телепорты, лимит 1 телепорт/тик
- Глаза следят за мышью (до старта, пауза, экраны game over/win)
- Хелп: циклическая карусель с canvas-анимациями
- Отзывы: 21 шт, фильтр по версии, пагинация
- Face-анимации: happy (победа + фейерверки), sad (1 проигрыш + слёзы), angry (2+ проигрыша + зубы + лазеры)
- High score в localStorage
- Закрытие попапов по Escape
- Анимация появления overlay (fadeIn)
- Мобильное управление: свайп по canvas, D-pad кнопки, тап для старта/автоигры
- Адаптивная верстка: canvas подстраивается под экран, touch-friendly кнопки
- Touch-трекинг глаз для бобра
